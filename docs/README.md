# Справочные материалы RadFlow

Эта папка — источник правды для сборки. Claude Code опирается на неё при разработке.
На деплой (Vercel) содержимое `docs/` не влияет.

- **plan/** — пошаговый план реализации MVP (главный документ; идите по нему).
- **architecture/** — архитектура системы и схема таблиц базы данных.
- **userflows/** — пользовательские воркфлоу: полный (`RadFlow_Aligned_v3`) и упрощённый (`СПРОЩЕНІ ..._v3`).
- **scenarios/** — каталог UX-сценариев всех ролей (`RadFlow_Catalog_v3`), детальный сценарий
  администратора (`User_Scenarios_Admin_Setup_Wizard.md`) и промпты лендинга (`RadFlow_Relume_Prompts.md`).
- **prototypes/screens/** — спецификации 11 экранов (дизайн-задания для каждого окна).
- **prototypes/code/** — рабочие HTML/JSX/CSS прототипы интерфейсов (визуальный референс).
- **prototypes/landing/** — прототип лендинга/регистрации.
- **diagram/** — SVG-диаграмма потока работы.

Все документы тех-стека приведены к актуальному решению: **Next.js + Supabase + Vercel + n8n Cloud**
(без отдельного бэкенда и без Railway).
