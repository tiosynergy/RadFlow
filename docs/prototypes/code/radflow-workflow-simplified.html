A. Початкове налаштування (5 кроків)

#ДіяЩо бачитьРезультат
1РеєстраціяНазва клініки, email, пароль — один екран.Акаунт створено.
2Профіль + прайс-листЗавантажити Word/Excel/PDF/посилання — AI-парсинг. Або вручну.Каталог готовий (fallback — ручна таблиця).
3Апарати та кабінетиНазва, тип МРТ/КТ, кабінет. «+ Додати ще».Кабінети збережені.
4Графік роботиДні + години. Шаблон Пн–Пт 8:00–18:00.Розклад збережено, Scheduler готовий.
5Персонал + Go LiveЗапросити радіолога. «Запустити кабінет».Кабінет активний.

⚡ Fallback: якщо AI не розпізнав прайс — проста таблиця для ручного введення.

B. Щоденна робота (3 кроки)

#ДіяЩо бачитьРезультат
1Дошка чергиУсі кабінети або один. Колір = статус. Календар + завантаженість + швидкий колл-лист праворуч.Видно, хто чекає / в кабінеті / пройшов.
2Call ListОбдзвін на завтра. Розгортання пацієнта (кабінет, тип, вік, дія). Підтверджений зникає зі швидкого списку.Завтрашня черга підтверджена.
3Новий запис / перезаписПошук або новий → слот. Інцидент → «Перенести день».Запис збережено.

⚡ Помилкове натискання статусу легко виправити: у розгорнутому рядку — сегмент «Змінити статус».

Радіолог
Черга · Картка пацієнта · Статус
#ДіяЩо бачитьРезультат
1Відкрити чергу«Моя черга» — лише авторизовані кабінети. Час, тип, пріоритет. CITO — першим.Бачу навантаження.
2Фільтр за кабінетомКлік по кабінету → лише його пацієнти.Сфокусована черга.
3Картка пацієнтаКлік по рядку → модальне вікно з усією клінічною інформацією + телефон направника.Повний контекст.
4Змінити статус + приміткаСегмент: Очікує / В кабінеті / Виконано / Не відбулось. Таймер. Нотатка (автозбереження).Статус синхронізовано з адміном.

⚡ Один екран: черга + картка пацієнта. Без перемикань між розділами.

2ДеталізаціяКлік на картку → таблиця по апаратах/послугах → «Скачати CSV».

Referral Portal · Запис · Статус
#ДіяЩо бачить

2Вибрати часКалендар вільних слотів → підтвердити. Email-підтвердження.

2. Ролі та доступ — коротко

РольЩо може
АдміністраторПовний доступ до кабінету. Запрошує персонал. Налаштовує все.
РадіологСвоя черга (авторизовані кабінети). Змінює статус, пише примітки.
Завідувач відділенняУсе що радіолог + статистика відділення.
CEO / ВласникТільки Dashboard і звіти. Не бачить медичних даних.
Лікар-направникТільки Referral Portal. Ізольований від внутрішнього UI.

3. Технічний стек MVP (коротко)

ШарТехнологіяНавіщо
FrontendNext.js 14 (React)SSR + routing, один репозиторій для всіх ролей.
BackendNode.js + Fastify (TypeScript)Одна мова наскрізь.
База / Auth / Storage / RealtimePostgreSQL (Supabase)RLS = мультитенант; WebSocket для Дошки черги.
AI-парсингGPT-4o (тільки онбординг)Прайс-лист → каталог. ~$0.01/запит.
Автоматизаціяn8n (6 workflows)Emails, invites, Call List, інциденти — без кастомного коду.
Хостинг / моніторингVercel · Railway · SentryDeploy з git push. ~$55–95/міс на 1–10 клінік.
